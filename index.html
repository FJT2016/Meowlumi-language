Language Rules

Plurals
Add -umi to any noun → flurr → flurrumi
Questions
Start with Nya? → Nya? Kesh flurr? = Are you good?
Past Tense
Add purr- before verb → purr-miru = went
Future Tense
Add mew- before verb → mew-miru = will go
Negation
Place neko before word → neko flurr = not good
Affection
Add -lumi suffix → keshlumi = dear you
Emphasis
Repeat first syllable → flu-flurr = wonderful
Word Order
Subject → Verb → Object (same as English)

Sample Phrases

Mrowlo! Nya kesh flurr?
"Hello! How are you?"
Mrowlo = Hello | Nya = question | kesh = you | flurr = good
Mew lumashi kesh, keshlumi.
"I love you, dear one."
Mew = I | lumashi = love | kesh = you | keshlumi = dear you
Nurrle flurr, shimura. Purrfin!
"Sleep well, friend. Goodbye!"
Nurrle = sleep | flurr = well | shimura = friend | Purrfin = goodbye
Wemi mew-miru lumara mewshin-nura.
"We will go home tomorrow."
Wemi = we | mew-miru = will go | lumara = home | mewshin-nura = tomorrow
